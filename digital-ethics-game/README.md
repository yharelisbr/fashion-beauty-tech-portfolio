# Digital Ethics Board Game (Genially)

An interactive digital board game created for a Philosophical Ethics course.  
Built using Genially, this project uses branching narratives, visual storytelling, and UI elements to guide players through ethical decision-making scenarios.

### Project Features
- Interactive, choice-based gameplay  
- Custom UI elements and layout  
- Ethical dilemma storytelling  
- Visual design and animation effects  
- Gamified learning approach  

### Skills Demonstrated
Interactive media • UI/UX concepts • Narrative design  
Visual storytelling • Digital educational tools
